API login-jwt

Rota Publica Get: https://login-jwt-79k0.onrender.com/

Retorno esperado:

                {
                    "msg": "HELLO WORLD!"
                }

.Endipoint de cadastro de usuarios: auth/register


Parametros:

  • "name"
  • "mail"
  • "password"
  • "ConfirmPassport"

.Exemplo

(URL): https://login-jwt-79k0.onrender.com/auth/register

                {
                    "name": "usuario",
                    "mail": "usuario@gmail.com"
                    "password": "segurança123"
                    "ConfirmPassport":"segurança123"
                }

Retorno esperado:

                {
                    "msg": "Email já registrado. Por favor utilize outro!"
                }

Para acessar a rota privada, precisa passar um id como parametro, exemplo:

https://login-jwt-79k0.onrender.com/user/66a92f50f89551a89c835040

Como é uma rota privada, não esqueça de Passar o Token do usuario registrado como Autorização.
